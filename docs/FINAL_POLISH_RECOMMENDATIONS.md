# Final polish recommendations

These are optional improvements that can make CareerWave feel more complete.

## Highest impact

1. Add real persistent file storage.
   - Replace local uploads with Vercel Blob, Supabase Storage, UploadThing, or S3.

2. Add email notifications.
   - Send emails when an application is accepted/rejected.
   - Send emails to employers when a new applicant applies.

3. Improve company profiles.
   - Add logo upload.
   - Add company public page.
   - Add list of all active jobs from that company.

4. Add admin dashboard.
   - Verify employers.
   - Moderate jobs.
   - View platform statistics.

5. Add Bulgarian language support.
   - Since the inspiration is jobs.bg, bilingual Bulgarian/English support would make the project more realistic.

## UI polish

- Add smoother dashboard charts.
- Add better mobile filters drawer.
- Add saved search filters.
- Add stronger empty states with call-to-action buttons.
- Add application timeline component.
- Add company logo fallback avatars.

## Product features

- Candidate matching score.
- Employer notes per candidate.
- Interview calendar integration.
- Job alert subscriptions.
- Public candidate portfolio links.
- Export applicants as CSV.

## Technical polish

- Add integration tests for APIs.
- Add Playwright end-to-end tests.
- Add CI pipeline for `typecheck`, `lint`, and `build`.
- Add database backups.
- Add structured logging.
- Add monitoring/error tracking.

## Deployment polish

- Add preview environment seed data.
- Add staging database separate from production.
- Add Vercel Analytics.
- Add custom 404 and error pages.
