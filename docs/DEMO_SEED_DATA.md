
```bash
npm run db:seed
```

## Seeded flows

The seed includes:

- active jobs
- job seeker profiles
- employer company profiles
- default CV documents
- submitted/reviewing/accepted applications
- bookmarks
- notifications
- an interview note

This gives enough data to demo both the job seeker dashboard and employer dashboard immediately after seeding.
